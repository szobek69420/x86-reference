VPCMPB - Packed CoMPare Byte (imm8=5)

VPCMPB k1{k2}, xmm2, xmm3/m128, 5    (V5+BW+VL
__mmask16 _mm_cmpge_epi8_mask(__m128i a, __m128i b)
__mmask16 _mm_mask_cmpge_epi8_mask(__mmask16 k2, __m128i a, __m128i b)

For each signed BYTE, if (1) >= (2) set 1, else set 0, to the corresponding bit of (3).
If k2 bit is 0, the comparison is not done and the corresponding bit of (3) is set to zero. Upper bits of (3) are zero cleared.
