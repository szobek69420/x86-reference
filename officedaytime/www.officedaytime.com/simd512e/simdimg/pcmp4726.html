PCMPGTD - Packed CoMPare Greater Than Dword

PCMPGTD xmm1, xmm2/m128    (S2
__m128i _mm_cmpgt_epi32(__m128i a, __m128i b)

For each signed DWORD, if (1) > (2) set 1, else set 0, to all bits in the corresponding DWORD of (3).

VPCMPGTD k1{k2}, xmm2, xmm3/m128/m32bcst    (V5+VL
__mmask8 _mm_cmpgt_epi32_mask(__m128i a, __m128i b)
__mmask8 _mm_mask_cmpgt_epi32_mask(__mmask8 k2, __m128i a, __m128i b)

For each signed DWORD, if (1) > (2) set 1, else set 0, to the corresponding bit of (3).
If k2 bit is 0, the comparison is not done and the corresponding bit of (3) is set to zero. Upper bits of (3) are zero cleared.
